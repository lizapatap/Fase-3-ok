Contendio:

  • HTML según nombra (Wikipedia®, 2020), siglas en inglés de HyperText Markup Language (‘lenguaje de marcas de hipertexto’), hace referencia al lenguaje de marcado para la elaboración de páginas web, de igual forma se considera un estándar que sirve de referencia para todo tipo de software que conecta o se relacione con la elaboración de páginas web en sus diferentes versiones, también define una estructura básica y un código (denominado código HTML), podemos incluir texto, imágenes, videos, juegos, entre otros; También es un estándar a cargo del World Wide Web Consortium (W3C) o Consorcio WWW, la cual es una de las más importantes organizaciones dedicada a la estandarización de casi todas las tecnologías ligadas a la web, pero enfocándose sobre todo en lo referente a su escritura e interpretación.
  • HTML se considera el lenguaje web más importante siendo su invención crucial en la aparición, desarrollo y expansión de la World Wide Web (WWW).
